Cheat Sheet — Python

Core concept
Python provides a structured way to solve problems efficiently.
Time complexity
Best: O(1) Average: O(n log n) Worst: O(n²)
Key methods
.init() .process() .validate() .transform() .output()
Use when
Problem fits the pattern, data is structured, performance matters.
Avoid when
A simpler solution exists or you are prematurely optimising.
Pro tip
Always handle edge cases: empty input, single element, max bounds.
